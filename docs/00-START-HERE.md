# START HERE — What Is This?

> Written for someone who has never used Claude Code, never heard of Anthropic, and has never used a terminal. If any of that describes you, you are in the right place.

## In one sentence

This is an upgrade pack for a program called **Claude Code** that turns it from a basic AI coding assistant into the most powerful AI development environment on the planet.

## I don't know what any of those words mean

That's fine. Let's go word by word.

### 1. What is Claude?

**Claude** is an AI made by a company called **Anthropic**. Think of it as a very smart assistant that you can talk to through typing. Like ChatGPT, but made by a different company. Many developers think Claude is the best AI for writing code.

### 2. What is Claude Code?

**Claude Code** is a program you install on your computer. It is like a chat window for Claude that lives in your **terminal** (see below). The difference from chatting with Claude in a browser: Claude Code can read your files, write new files, run programs, test things, and complete real work on your computer. It is Claude with hands.

### 3. What is a terminal?

The **terminal** is a text window on your computer where you can type commands instead of clicking buttons. It sounds scary but it is just a text box. On Windows it is called "Command Prompt" or "PowerShell". On Mac it is called "Terminal". We'll show you how to open it in the next tutorial.

### 4. What is Claude Apex?

**Claude Apex** (this repository) is a big upgrade pack for Claude Code. It adds:
- 25 specialist AI helpers (called **agents**)
- 1,276+ pre-written playbooks Claude reads when needed (called **skills**)
- Automatic routing that makes Claude smart about which tools to use
- Documentation that you are reading right now

When you install Apex, your Claude Code goes from "okay" to "very, very capable".

## Why would I want this?

If you want to:
- Build websites without being a web developer
- Write code faster than you thought possible
- Have a pair programmer who never gets tired
- Do research, write documents, design graphics, test software — all by describing what you want

This repository is for you. It gives you the exact same setup that professional developers use.

## Will my computer explode?

No. Here is what happens when you install Apex:

1. It makes a backup of everything in your existing `~/.claude/` folder
2. It adds new files next to your old ones — never overwriting anything
3. If something goes wrong, you run `uninstall.sh` and everything reverts

It is as safe as installing any other software.
