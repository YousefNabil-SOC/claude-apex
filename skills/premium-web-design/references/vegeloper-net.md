# Vegeloper (Carlos Enrique Serrano Varela) - Design Reference

**URL:** https://vegeloper.net/
**Analyzed:** 2026-04-23
**Overall Rating:** 3.5/5

## Tech Stack
- Animation: GSAP + JavaScript (mentioned on page)
- 3D Graphics: Blender and Autodesk Maya integration
- Framework: HTML5 + CSS3 + JavaScript
- Navigation: Anchor-based (#sitiosweb, #sobremi)

## Hero Section
Classic portfolio hero featuring large typographic layout with developer name ("CARLOS ENRIQUE SERRANO VARELA") as primary focal point, paired with professional subtitle: "Senior UX/UI / Frontend Developer." Name acts as visual anchor before case studies.

## Navigation
Fixed navigation with primary links: Proyectos, Sobre mi, and Contacto. Anchor-based navigation using hashtags (#sitiosweb, #sobremi) enables smooth scrolling to sections. Direct WhatsApp contact link (wa.me/522311285874) provides alternative engagement path.

## Scroll Animations
GSAP animations drive scroll effects. Likely includes staggered reveals of portfolio items as user scrolls. 3D capabilities via Blender/Maya suggest complex motion graphics beyond standard CSS animations.

## Hover & Interactions
Interactive elements include:
- "Explorar" buttons linking to live projects
- Figma prototype embeds for interactive showcases
- App store links for mobile applications
- Social media CTAs with hover states

## Typography & Color
Modern, minimalist design centered on project showcases rather than ornamental visuals. Clear sectional organization with H2-level headings for portfolio categories (Sitios Web, Apps moviles, Aplicaciones web, Sobre mi). Emphasis on project imagery over decorative typography.

## Page Transitions
Anchor-based navigation with smooth scroll transitions between sections. No route-level transitions evident. Focus on within-page navigation experience.

## Unique Elements
Technology stack display using vendor icons (HTML5, CSS3, JavaScript, PHP, React, Angular, MySQL, PostgreSQL, GitHub, WooCommerce, Figma, Adobe Suite). TikTok presence integration suggests content creator positioning. 3D motion graphics via Blender/Maya integration.

## Reusable Patterns
1. Large typographic hero with developer name (personal branding)
2. Fixed navigation with anchor links for section jumping
3. Technology stack visualization with vendor icons
4. Figma prototype embeds for live interaction demonstrations
5. App store links for mobile portfolio items
6. GSAP-driven scroll animations for portfolio reveals
7. WhatsApp contact integration for direct messaging
8. Social media footer with TikTok/Instagram presence

## Anti-Patterns to Avoid
- Avoid purely image-based portfolio when project links are available
- Over-reliance on 3D graphics when 2D designs communicate faster
- Inconsistent contact methods (mix of WhatsApp, Figma, app store links)
