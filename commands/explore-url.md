Explore, analyze, and summarize a website or URL using the Playwright MCP browser.

The user will provide a URL.

Steps:
1. Use Playwright MCP to navigate to the URL
2. Capture a screenshot
3. Extract and analyze:
   - Page title, meta description, main heading
   - Key content sections and their purpose
   - Technology stack (check for React, Next.js, etc. in page source)
   - Navigation structure and site map hints
   - Contact information, pricing, services offered
   - Performance notes (load indicators, images, fonts)
4. For competitor analysis: identify strengths, weaknesses, design patterns
5. For research: summarize key information in structured format
6. For accessibility: note any obvious a11y issues (missing alt text, contrast, etc.)

Output: structured markdown summary with screenshot reference.
